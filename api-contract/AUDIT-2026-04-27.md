## Top 3 Ambiguities / Issues for Plan8 Codex

### 1. **Login/Logout Prefix Mismatch**
- **Spec requires:** `/uiadapter/login.do`, `/uiadapter/logout.do`
- **Jakarta has:** `/login.do`, `/logout.do` (missing `/uiadapter/` prefix)
- **Javax has:** `/login`, `/logout` (missing BOTH `/uiadapter/` prefix AND `.do` suffix)
- **Flag for Phase D:** LoginController needs @RequestMapping("/uiadapter") at class level + both methods need `.do` suffix in javax

### 2. **HTTP Verb Mismatch: GET vs POST**
- **Spec mandates:** All 14 endpoints are POST
- **Current code has:** UiadapterFileController (GET), UiadapterVideoController (GET)
- **Note:** These are file/stream operations, which *semantically* use GET (query params, range requests). Spec may need revision or controllers must be corrected.
- **Flag for Phase D:** Clarify if spec #9 (advancedDownloadList.do) is GET or POST before fixing

### 3. **Endpoint Path Naming Divergence**
- **Jakarta path:** `/uiadapter/multiDownloadFiles.do` (not in spec)
- **Spec path:** `/uiadapter/advancedDownloadList.do` (not found)
- **Same semantic purpose?** Both are multi-file download, but names don't match.
- **Flag for Phase D:** Verify if `multiDownloadFiles` is alias for `advancedDownloadList` or separate feature. If alias, rename controller method.
